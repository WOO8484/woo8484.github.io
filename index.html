개인용 연결 웹 테스트

최신 주소를 저장하고 아이폰 Safari에서 바로 여는 간단한 시험판입니다.

아이폰에서 시험하는 순서

  1. 이 페이지를 Safari로 엽니다.
  2. 공유 → 홈 화면에 추가합니다.
  3. ‘사이트 열기’로 원본 사이트에 접속합니다.
  4. Safari 북마크 하나를 만든 뒤 주소를 복사한 스크립트로 교체합니다.
  5. 사이트에서 해당 북마크를 실행해 배너 제거 여부를 확인합니다.

현재 시험판의 한계

GitHub Pages는 다른 사이트 내부를 직접 수정할 수 없습니다. 배너 정리는 Safari에서 실행하는 개인용 스크립트가 담당합니다. 사이트 구조에 따라 일부 요소는 남을 수 있습니다.
