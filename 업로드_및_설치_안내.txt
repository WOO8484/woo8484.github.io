[중요]
이 ZIP을 GitHub 저장소에 ZIP 파일 그대로 보관하는 것이 아니라, 압축을 풀어 나온 파일과 폴더 전체를 저장소 루트에 업로드해야 합니다.

저장소 루트에서 다음 항목이 바로 보여야 합니다.
- .github
- app
- build.gradle.kts
- settings.gradle.kts
- gradle.properties
- README.md

빌드 절차
1. GitHub 저장소 상단 Actions
2. Android APK 자동 빌드
3. Run workflow
4. 초록색 완료 확인
5. 완료된 실행 화면 하단 Artifacts
6. GPT-공작소-네이버도우미-debug-apk 다운로드
7. 압축 해제 후 app-debug.apk 설치

설치 전
- 기존 GPT 공작소 네이버 도우미가 있다면 삭제
- 설정 > 보안 및 개인정보 보호 > 기타 보안 설정 > 출처를 알 수 없는 앱 설치에서 사용 중인 브라우저 또는 내 파일 허용

설치 후
- 앱 정보 > 우측 상단 점 3개 > 제한된 설정 허용(표시되는 기기만)
- 설정 > 접근성 > 설치된 앱 > GPT 공작소 네이버 자동입력 허용
