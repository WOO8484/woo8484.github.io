GPT 공작소 네이버 도우미 v1.0.2 접근성 없는 설치 검증판

1. 이 ZIP을 압축 해제합니다.
2. 내부 파일과 폴더 전체를 GitHub 저장소 루트에 덮어씁니다.
3. 삭제 파일도 반드시 반영합니다.
   - app/src/main/java/com/gptgongjakso/naverhelper/NaverAccessibilityService.java
   - app/src/main/java/com/gptgongjakso/naverhelper/AutomationStore.java
   - app/src/main/res/xml/accessibility_service_config.xml
4. GitHub Actions → Android APK 자동 빌드 → Run workflow를 실행합니다.
5. 성공 후 Artifact ZIP을 내려받아 app-debug.apk를 설치합니다.

이 버전은 접근성 서비스를 포함하지 않으므로 자동입력 기능은 없습니다.
