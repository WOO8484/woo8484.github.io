# GPT 공작소 네이버 도우미

이 저장소는 GitHub Actions에서 설치 검증용 Android Debug APK를 자동 빌드하도록 구성되어 있습니다.

## GitHub에 올리는 방법

이 ZIP의 압축을 푼 뒤 `app`, `.github`, `settings.gradle.kts` 등이 저장소 최상단에 보이도록 전체 파일을 업로드합니다. ZIP 파일 자체만 저장소에 올리면 자동 빌드되지 않습니다.

## APK 만드는 방법

1. 저장소의 `Actions` 메뉴를 엽니다.
2. `Android APK 자동 빌드`를 선택합니다.
3. `Run workflow`를 누릅니다.
4. 빌드가 초록색으로 완료되면 실행 화면 아래 `Artifacts`에서 `GPT-공작소-네이버도우미-debug-apk`를 내려받습니다.
5. ZIP을 풀고 `app-debug.apk`를 설치합니다.

## 설치 실패 예방

- 예전에 설치한 같은 패키지 앱이 있으면 먼저 삭제합니다.
- 휴대폰에서 브라우저 또는 내 파일 앱의 `이 출처 허용`을 켭니다.
- Android 13 이상에서 접근성 설정이 제한되면 앱 정보의 우측 상단 메뉴에서 `제한된 설정 허용`을 먼저 선택합니다.
- Debug APK는 Gradle이 자동 서명하므로 첫 설치 테스트에 적합합니다.

## 기능 범위

- v5.3 규격 ZIP에서 `naver_metadata.json`, `naver_content.txt` 읽기
- 자동 발행 허용 패키지 차단
- 접근성 설정 열기
- 네이버 블로그 글쓰기 페이지 열기
- 제목과 본문 자동입력 시도
- 즉시 중지
- 발행·임시저장 자동 클릭 금지

현재 버전은 설치 및 제목·본문 자동입력 검증판입니다. 네이버 편집기 구조에 따라 이미지, 게시판, 태그 기능은 실제 기기 테스트 후 별도 보정이 필요합니다.
