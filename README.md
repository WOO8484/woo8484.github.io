# GPT 공작소 네이버 도우미 v1.0.2

접근성 서비스를 완전히 제거한 APK 설치 검증판입니다.

## 포함 기능
- Android APK 설치 및 실행
- `naver_metadata.json`, `naver_content.txt` 포함 ZIP 검사
- 제목·본문·게시판·태그·이미지 수 표시
- 네이버 블로그 글쓰기 페이지 열기

## 제외 기능
- 접근성 서비스 및 관련 권한
- 화면 자동입력
- 게시판·이미지·태그 자동조작
- 임시저장·발행 자동 클릭

GitHub Actions의 `Android APK 자동 빌드`를 실행하고 Artifact에서 `app-debug.apk`를 받으세요.
